Automação de Tarefas com Gulp

Nessa aula aprendemos a usar o Gulp para automatizar tarefas

Gulp-Sass

Aprendemos a automatizar a importação dos arquivos Sass

Também conseguimos comprimir o arquivo main.css

Gulp-JavaScript

Aprendemos a compilar os arquivos de JavaScript

Também aprendemos como usar a Obfscate para tornar o arquivo ilegível

Gulp-Images

Aprendemos a deixar as imagens mais leves usando o ImagesMin
